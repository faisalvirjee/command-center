Security

Known App Files Registry

Files not on this list are flagged Unknown during a scan.

How to register a new app file:
Edit drive-vault.html and find the FILE_LABELS constant. Add a new entry like:

'my_new_app.json': { app: 'My New App', status: 'active' }
Valid status values: 'active' or 'archived'. Re-deploy and the file will be recognized in future scans.

Shared Folder Targets

My Drive paths scanned for shared app data. Add a new entry to SHARED_TARGETS as apps grow.

How to add a new Shared Folder Target:
Edit drive-vault.html and find the SHARED_TARGETS constant. Add a new entry like:

{ app: "LPL Tracker", label: "LPL Shared", path: ["ClaudeApps","LPL_Shared"] }
Re-deploy and the new path will appear here and be included in all future scans.
